🗄️ Configurar base de datos

Pasos rápidos (gratis):
1. Ve a supabase.com → Start your project
2. Nuevo proyecto → espera ~2 min
3. SQL Editor → pega y ejecuta:
CREATE TABLE epelsa_data (key text PRIMARY KEY, value jsonb, ts timestamptz DEFAULT now());
4. Settings → API → copia Project URL y anon key

💡 Al guardar, se te ofrecerá descargar el HTML con las credenciales integradas. Sube ese archivo a GitHub para que los datos no se pierdan al borrar caché.
